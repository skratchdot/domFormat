domFormat Bookmarklet

This is a tool for viewing the formatted source of a page. To install it, drag the "View Source" link below to your bookmarks toolbar. Visit any webpage, and click the bookmark you just saved.

-- View Source --

All Bookmarklets:

View Source (pretty)
This is the main bookmarklet. It allows you to view the formatted source as well as the original source in an editor with syntax highlighting.

View Source (plain)
This bookmarklet shows the formatted source in a <textarea /> element. It is quick, but does not have the extended options of the main bookmarklet.

Reload Formatted
This reloads the page with the formatted source.
